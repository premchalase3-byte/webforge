WebForge — AI Website Builder

«Forge the impossible. Build full websites instantly using AI.»

---

 Overview

WebForge is an AI-powered website builder that transforms simple text prompts into fully functional web applications.
Users can describe their idea, and WebForge generates a complete project with UI, code, and live preview — instantly.

It combines AI, modern web technologies, and an interactive robot assistant to create a seamless development experience.

---

Features

-  AI Website Generation
  Generate complete websites from natural language prompts

-  Instant Live Preview
  See your website in real-time with code + preview mode

- Interactive Robot Mascot
  Human-like assistant that explains what's happening

- Prompt Enhancement (AI)
  Improves your idea before building

- Responsive Design
  Mobile + desktop ready layouts

- 3D Website Showcase Carousel
  Explore generated websites visually

- Authentication System
  Secure login with Google

- 📦 Download Full Project
  Export complete working code

---

How It Works

1. User enters a website idea
2. AI enhances the prompt
3. WebForge generates structured code
4. Live preview is rendered instantly
5. User can edit, explore, and download


 Robot Mascot Experience

WebForge includes a smart AI mascot that behaves like a developer partner:

- “Hmm… let me think about this 👀”
- “I’m building your components…”
- “Optimizing your layout for mobile 📱”
- “Done! Take a look 🚀”
